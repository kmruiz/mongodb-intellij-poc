Detects queries that potentially impact cluster performance due to not using proper indexes.

Queries not using proper indexes force MongoDB to run a COLLSCAN. COLLSCAN stages loads all your data from disk into memory, and filters documents one by one. This means that they are the most expensive queries and should be avoided.

Consider creating an index to cover this query following indexing best practices.
